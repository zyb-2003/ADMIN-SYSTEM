# 后台管理系统

## 项目介绍
一个完整的后台管理系统，包含登录认证、用户管理、商品管理、数据仪表盘等功能。

## 技术栈
- Vue 3 - 前端框架
- Pinia - 状态管理
- Vue Router - 路由管理
- ECharts - 数据可视化
- Vite - 构建工具
- LocalStorage - 数据持久化

## 核心功能
1. **登录认证** - 用户登录、token持久化、路由守卫
2. **用户管理** - 用户列表增删改查、搜索筛选、批量删除
3. **商品管理** - 商品列表增删改查、分类筛选、库存预警
4. **仪表盘** - 数据统计卡片、ECharts图表展示、低库存预警
